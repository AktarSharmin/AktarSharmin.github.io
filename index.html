Research

  • Developed a contractive autoencoder-based deep learning model for intrusion detection, achieving a peak accuracy of 97.58%.
  • Created stateful and stateless RNN models for sentiment-based text generation.
  • Leveraged machine learning to reverse-engineer SQL queries from database memory, achieving an accuracy of 92% for MySQL and 90% for PostgreSQL.
  • Designed a machine learning-based system to detect anomalies in robotic system logs, enhancing safety and reliability.
  • Implemented a parallel version of the K-means clustering algorithm for intrusion detection in a big data environment.
